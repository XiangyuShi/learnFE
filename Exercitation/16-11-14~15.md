#Exercitation

1.cortex项目构建过程：
初始化项目
git clone project
cd project
git checkout branch
npm i
cortex install
cortex build --prerelease beta
cortex server
2.
git branch  查看当前分支
git branch name 新建name分支
git checkout name  切换到name分支
git add .  添加所有修改
git status 查看文件状态
git commit -m ‘xxx’  描述修改
git push origin name 推送到远端name分支
然后在CI建立模拟环境，测试修改。确认无误后，向master分支提交代码
